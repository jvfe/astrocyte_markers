scRNA-seq Analysis

Selected datasets

  • PRJNA408230 (DropSeq) - 2020
    • Pallium expression in Pogona vitticeps (lizard species)
    • 35 samples
  • GSE150486 (10x) - 2020
    • Gene expression in two song-production related regions in Lonchura striata domestica (Bengali Finch)
    • 4 samples
  • GSE230116 (10x) - 2023
    • Astrocyte expression in mice (only WT samples will be used)
    • 4 samples

Methodology

  • Data was be acquired from SRA through the nf-core/fetchngs pipeline.

  • Count matrices were assembled through the nf-core/scrnaseq pipeline, using kallisto for alignment.

  • Matrices were then processed in the R programming language, using the Seurat package.

    • The integrated dataset was then annotated using TransferData to a Mus musculus reference expression dataset.

  • All computational infrastructure was provided by NPAD/UFRN.

Results

  • The only dataset in which Astrocytes were detected among the cell clusters was the Mus musculus dataset.
  • Annotation results for L. striata and P. vitticeps showed a significant number of Neurons annotated, and multiple populations of neurons.
    • This is consistent with the processed results from the P. vitticeps study, which showed no Astrocyte population, only different neuron subtypes.
